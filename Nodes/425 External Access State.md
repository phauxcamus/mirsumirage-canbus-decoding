# `425` External Access State

## Data
### Data 1 (8 Bytes)
Bytes:
1. Constant `00`
2. Door State
	- `00`: All closed
	- `01`: Front Right
	- `03`: Front Left
3. Lock State
	- `01`: Locked
	- `02`: Unlocked
4. Constant `00`
5. Constant `00`
6. Constant `00`
7. Constant `00`
8. Constant `00`

## Notes
Door and Lock states are not fully mapped yet
